Домашняя страница

Инструкция к боту:

Это простая страница телеграмм бота, написанного на Django + Aiogram.

Через Django Admin можно добавлять и удалять категории и товары. Используется PostgreSql.

В самом телеграмм боте можно увидеть эти категории и сами товары.

Есть также команды, такие как "помощь", "описание", "каталог".

Нажмите на меня, чтобы перейти в Django Admin, или, добавь в конец url-адреса: /admin/
